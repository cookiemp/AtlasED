Which Python data structure is best suited for storing unique elements and provides O(1) lookup time?

Context 
# You need to store student IDs and check existence quickly
student_ids = ?

# Check if student exists
if student_id in student_ids:
    print("Student found")

Correct!

A Set is the correct choice because it stores only unique elements and provides O(1) average time complexity for lookup operations. Lists and tuples have O(n) lookup time, while dictionaries are designed for key-value pairs rather than just storing unique values.
