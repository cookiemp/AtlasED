# AtlasED Design System

## Product Context

AtlasED is a **desktop learning platform** that transforms YouTube playlists into structured, interactive courses. It combines video learning with AI-generated notes ("Field Guides"), comprehension quizzes, and an AI tutor ("Compass AI").

### Core Value Proposition
- Convert passive YouTube watching into active learning
- AI-generated study materials from video transcripts
- Spaced repetition for long-term retention
- Interactive quizzes for comprehension

### Target Users
- Self-directed learners
- Technical professionals upskilling
- Students supplementing coursework
- Lifelong learners

## Key Pages & Architecture

1. **Dashboard** - Entry point, expedition management
2. **Expedition View** - Course overview, waypoint list
3. **Video Player (Cockpit)** - Core learning interface
4. **Settings** - Configuration and API keys

### Metaphors
- **Expedition** = Course/Playlist
- **Waypoint** = Video/Lesson
- **Field Guide** = AI-generated notes
- **Charted** = Completed
- **Compass AI** = AI tutor/chat

## Current Visual Design

### Color Scheme
- **Primary:** Gold (#d4af35)
- **Background:** Deep charcoal (#16140d)
- **Panels:** Warm dark (#262319)
- **Borders:** Gold-tinted (#433d28)
- **Text:** White → Beige → Muted

### Typography
- **Display:** Inter (UI elements)
- **Serif:** Merriweather (titles, emphasis)
- **Mono:** JetBrains Mono (metadata, code)

### Current Pain Points (for redesign)
1. Dashboard feels sparse and utilitarian
2. No clear visual hierarchy in expedition cards
3. Video player (Cockpit) is functional but not "delightful"
4. Settings page is basic form layout
5. Missing visual polish and micro-interactions
6. Could benefit from more "exploration/adventure" theme integration

## UX Patterns

### Navigation
- Fixed title bar at top
- Simple icon-based nav (Home, Settings)
- Back buttons for deep navigation
- Breadcrumb-like expedition title display

### Layout
- Responsive but desktop-focused
- Max-width containers for readability
- Three-column layout for Cockpit
- Card-based organization

### Feedback
- Toast/snackbar style messages (limited)
- Button loading states
- Progress indicators
- Hover effects on interactive elements

## Design Goals for Overhaul

1. **Elevate the "Expedition" theme** - More adventure/map/compass visual language
2. **Improve information hierarchy** - Clearer visual distinction between elements
3. **Add delight** - Micro-interactions, animations, satisfying feedback
4. **Polish the Cockpit** - This is the core value, make it shine
5. **Better empty states** - More inviting first-time experience
6. **Enhanced dashboard** - Make expeditions feel like achievements/progress
7. **Consistent spacing** - Tighter, more intentional layout
8. **Modern touches** - Subtle gradients, glass effects, depth

## Brand Personality

- **Adventurous** but **focused**
- **Premium** but **accessible**
- **Smart** but **not pretentious**
- **Guided** but **self-directed**
